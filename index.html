A) Introduccion A) Introduction

El Sistema de Gestion de Gimnasios para CakePHP es la solucion integral para administrar todas las operaciones de un gimnasio. El sistema implementa diferentes roles de usuario como Personal (Staff Member), Miembro (Member), Contador (Accountant) y Administrador (Admin).

Esta guia de usuario cubre todas las opciones que proporciona este script de CakePHP. Se recomienda leer la documentacion detenidamente ya que probablemente contiene respuestas a la mayoria de las preguntas que puedan surgir. Si considera que algun aspecto no esta cubierto o tiene consultas adicionales, contactenos via correo electronico a soporte@systaich.com.

Gym Management System for CakePHP is the ideal way to manage complete Gym operations. It has different user roles like Staff Member, Member, Accountant and Admin user.

G) Preguntas Frecuentes (FAQ) G) Frequently Asked Questions (FAQ)

1) I Don't see Installer Page properly. No background image, no Next button nothing.

Asegurese de tener el archivo .htaccess en el directorio principal (en la carpeta gym_master).

Make Sure you have .htaccess file on main directory (in gym_master folder).

Si ya tiene el archivo .htaccess, asegurese de que su host tenga el modulo mod_rewrite activado.

If you already have .htaccess, make sure your host has mod_rewrite module on.

2) When I go to Payment pages it says: An Internal Error Has Occurred.

Para esto debe volver a cargar el archivo paypal_class.php via cPanel.

For this you have to re-upload file paypal_class.php via cPanel file manager.

Si desea personalizacion paga, contactenos en soporte@systaich.com. If you want paid customization, contact us at support@systaich.com.
