A) Introduccion A) Introduction

El Sistema de Gestion de Gimnasios para CakePHP es la solucion integral para administrar todas las operaciones de un gimnasio. El sistema implementa diferentes roles de usuario: Personal (Staff Member), Miembro (Member), Contador (Accountant) y Administrador (Admin).

Esta guia cubre todas las opciones del sistema. Se recomienda leer la documentacion detenidamente. Para consultas adicionales, contactenos en soporte@systaich.com.

Gym Management System for CakePHP is the ideal solution to manage complete Gym operations. It has different user roles: Staff Member, Member, Accountant and Admin user.

This guide covers all system options. Please read the documentation carefully. For additional questions, contact us at support@systaich.com.
